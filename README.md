# Customer Segmentation using K-Means Clustering

This is a mini Data Science and Machine Learning project that groups customers based on their spending behavior using the K-Means Clustering algorithm.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Features
- Data preprocessing
- Feature scaling
- Elbow method
- K-Means clustering
- Cluster visualization
- Customer segmentation analysis

## Dataset
Dataset contains:
- CustomerID
- Gender
- Age
- Annual Income
- Spending Score

## Machine Learning Algorithm
- K-Means Clustering

## Outputs
- Elbow Method graph
- Customer Segmentation graph

## Project Structure
- data/ → dataset
- src/ → Python source code
- outputs/ → saved graphs

## Objective
To segment customers into different groups based on income and spending patterns.
